# SPD Sterilizers

A React project about sterilizers used in the Sterile Processing Department.

## About
This website shares information about sterilizers, their purpose in sterile processing, and the importance of proper sterilization for patient safety.

## Features
- About Me section
- Contact / survey form
- Sterilizer project information
- Images related to sterile processing

- EmailJS
